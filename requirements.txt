# 资产管理系统依赖要求

## 系统环境
- Node.js v14 or higher
- npm (Node Package Manager)

## Node.js 依赖包
express@^4.18.2
cors@^2.8.5
sqlite3@^5.1.6
qrcode@^1.5.3
uuid@^9.0.0
bcryptjs@^2.4.3
jsonwebtoken@^9.0.0
exceljs@^4.3.0

## 可选依赖（用于开发）
nodemon@^2.0.20  # 开发时自动重启

## 数据库
SQLite3 (已包含在 sqlite3 包中，无需额外安装)

## 浏览器兼容性
- Chrome 最新版
- Firefox 最新版  
- Safari 最新版
- Edge 最新版

## 移动端支持
- iOS Safari
- Android Chrome
